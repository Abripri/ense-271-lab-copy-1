## Background and Goals:

### Background
  * Local: Allows you to build a website.
  * Wordpress: Used to build and manage websites.
  * Persona: A fictional and realistic model of a target user.

### Goals
  * To create a persona for a person using a ticket purchasing app
  * To create a personal website for this persona using [Local](https://localwp.com)
  * To create a home page for the site
  * To create three static non-changing pages for the site
  * To create three blog posts written in the point of view of the persona
